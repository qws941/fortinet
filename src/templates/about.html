FortiGate 방화벽 경로 분석기 정보

FortiGate 방화벽 경로 분석기

버전 1.0.0

주요 기능
  • 출발지에서 목적지까지의 트래픽 경로 추적
  • 각 방화벽에서 적용되는 정책 규칙 상세 분석
  • 트래픽 경로의 시각적 다이어그램 생성
  • 차단된 트래픽의 정확한 차단 지점 및 규칙 식별
  • CSV 파일을 통한 다수의 트래픽 경로 일괄 분석
  • 오프라인 환경에서도 동작하는 독립 실행형 애플리케이션
  • FortiGate 방화벽과 FortiManager 통합 모두 지원
기술 정보
개발 언어 Python 3.9+
웹 프레임워크 Flask
UI 프레임워크 Bootstrap 5
시각화 라이브러리 Cytoscape.js
데이터 테이블 DataTables
배포 형태 Docker 컨테이너
연관 문서
사용자 도움말 배치 분석 샘플 CSV
관리자 정보
